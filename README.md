# PyPanel Browser IDE

**PyPanel** は、ブラウザ上で動作する統合開発環境（IDE）です。
サーバー構築不要で、Python, HTML, CSS, JavaScript などのコードを記述・実行できます。

> **🤖 AI開発プロジェクト**  
> このソフトウェアのソースコードおよびドキュメントは、**生成AI（Artificial Intelligence）** の支援によって作成されました。  
> ユーザーのフィードバック（スマホ操作の改善、ファイル管理機能など）をAIが解析し、コードを修正・実装しています。

---

## 📂 主な機能

- **プロジェクト実行**: `index.html` を起点に、リンクされたCSSやJSファイルを自動で読み込んで実行します。
- **ファイル管理**: 仮想ファイルシステムにより、ブラウザ内でファイルやフォルダを作成・編集・削除できます。
- **エディタ**: 高機能なエディタ（Monaco Editor）を採用し、シンタックスハイライトや全角スペースの可視化に対応しています。
- **スマホ対応**: タッチ操作でのターミナルリサイズや、モバイル向けのUI調整を行っています。

## 📱 操作方法（スマホ・PC共通）

### ファイル操作
サイドバーのエクスプローラーを使用します。
- **作成**: 上部の `+` (ファイル) や `📁` (フォルダ) アイコンをタップします。
- **メニュー**: 各ファイルの横にある `⋮` ボタンをタップすると、以下の操作が可能です。
  - **名前変更**: ファイル名を変更します。
  - **移動**: ファイルを別のフォルダへ移動します。
  - **削除**: ファイルを削除します。

### 実行
- **Run**: `▶ Run` ボタンで、現在のプロジェクトを実行します。
- **Preview**: `❐ Pop` ボタンで、全画面プレビューを表示します。

## ⚠️ 注意事項
- 本ツールはブラウザのローカルストレージを使用します。ブラウザのキャッシュをクリアするとデータが消える可能性があるため、重要なコードはこまめにバックアップ（コピー）してください。
